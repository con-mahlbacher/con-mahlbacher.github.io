College Soccer Elo Project

This is a simple python script that scrapes college soccer match data from the web and calculates elo ratings for every division 1 soccer team based on this data. At the moment, it prints out all the game results followed by a ranking with each team. I am currently working on putting this onto this website and improving the user display.
